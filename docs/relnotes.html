Release Notes

Mesa uses an even/odd version number scheme like the Linux kernel. Odd numbered versions (such as 7.5) designate new developmental releases. Even numbered versions (such as 7.4) designate stable releases.

The release notes summarize what's new or changed in each Mesa release.
